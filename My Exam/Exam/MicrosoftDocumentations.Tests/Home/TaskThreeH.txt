III. TEST CASES - C# DOCUMENTATION 
Create a test with Selenium WebDriver, which does the following:

III H. 
1. Navigate to https://docs.microsoft.com/en-us/dotnet/csharp/; 
2. Click on every "C# Guide" page's "In this Article" right navigation hyperlink;
3. Verify that every "In this Article" right navigation hyperlink scrolls the web page to an article section with the same name.

Expected results:
1. Every "C# Guide" page's "In this Article" hyperlink scrolls to an article section with the same name.
2. Every "C# Guide" page's article section is displayed on the screen.
3. Every "C# Guide" page's article section is represented in the HTML with <h2> tag.
